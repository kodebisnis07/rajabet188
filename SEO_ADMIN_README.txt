Raja Topup Games - SEO Admin Fix

Fitur yang sudah ada di paket ini:
- Route: /admin/seo
- Menu sidebar: SEO Website
- Template: app/templates/admin/seo.html
- Penyimpanan ke table settings memakai model Setting
- Integrasi meta SEO ke app/templates/base.html

Cara pakai:
1. Extract ZIP ini.
2. Masuk ke folder RAJATOPUPGAMES-TERMURAH2026.
3. Jalankan:
   git add .
   git commit -m "Add SEO admin page"
   git push origin main
4. Render -> Manual Deploy -> Deploy latest commit.
5. Buka https://rajatopupgames-termurah2026.onrender.com/admin/seo

Catatan:
Kalau masih 404 setelah deploy, berarti Render belum memakai commit terbaru atau Anda push dari folder yang salah.
